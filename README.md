# Программа для построения 2D-эскизов с возможностью наложения геометрических ограничений
Командный проект, нацеленный на разработку геометрического решателя - компонента CAD-систем, отвечающего за перестроение гоеометрии в соответствии с ограничениями
при добавлении новых ограничений или перетаскивании уже существующих геометрических объектов.
Для демонстрации работы решателя с помощью PyQt реализован пользовательский интерфейс с возможностью добавления в рабочее поле точек и отрезков и наложения 9 типов геометрических
ограничений (совпадание точек, фиксация точки, параллельность прямых, расстояние между точками, угол между прямыми, принадлежность точки прямой, перпендикулярность прямых,
вертикальность и горизонтальность прямой).

Основной файл - geometry.py реализует цикл обработки поступающих от интерфейсной части данных, отвечает за пересчет положения объектов, связанных с поступающими с изменениями для
избежания пересчёта всеё геометрии эскиза (что определяется с помощью графа геометрических связей).

Ограничения реализованы в geometry.py как функции, возвращающие в основной цикл обработки геометрии матрицы и вектора правых частей на основе посчитанных зараенне
аналитически выражений.

В качестве примера построена схема кривошипно-ползунного механизма (в правой части интерфейса отображаются актуальные геометрически ограничения), продемонстрировано
корректное изменение положения точек в соответствии с геометрией механизма.
